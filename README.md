# Retro Style Portfolio

A nostalgic and visually engaging portfolio website inspired by retro design aesthetics. This project showcases a unique take on personal branding using vintage design elements and classic web styling.

## Project Description

This retro-style portfolio is a personal website designed to display projects, skills, and contact information with a charming vintage aesthetic. The design draws inspiration from classic computer interfaces and 80s/90s web design trends, creating a memorable and distinctive user experience.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Build Tools**: Static site architecture
- **Assets**: Custom SVG graphics and media files
- **Development Tools**: Claude, Cursor (AI-assisted development)
- **Audio**: Web Audio API for retro sounds

## Features

- ✅ Responsive retro-inspired design  
- ✅ Smooth animations and interactions  
- ✅ Project showcase section  
- ✅ Contact information and social links  
- ✅ Vintage UI elements and color schemes
